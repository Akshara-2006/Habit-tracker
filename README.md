
# Habit Tracker PWA

## Features
- Add / delete habits
- Daily check-ins
- Streak tracking
- LocalStorage persistence
- Progress charts
- PWA setup
- Offline-ready structure

## Run Project

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
